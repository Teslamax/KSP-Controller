# KSP Controller Project

A USB-HID and network-enabled controller for Kerbal Space Program using the Adafruit ESP32-S3 Reverse TFT Feather.

## Features
- USB Keyboard HID (working)
- Gamepad HID (planned)
- Dual CDC: Debug + Simpit (planned)
- SD-based config
- NeoKey + GPIO input
- TFT display
- RTC + NTP time sync
- Ethernet networking
